##Les Sélecteurs Utile: http://code.tutsplus.com/tutorials/the-30-css-selectors-you-must-memorize--net-16048
##### Exemple <button type="submit" class="btn btn--submit" id="bouton-y" name="Envoyer"></button>
##### Tag button{ background-color: #e67e22; font-size: 20px; }
##### Attributs [type="submit"]{ border: none; }
##### Class .btn--submit{ color: white; }
##### Class attributs [class*="btn"]{ padding: 10px; }
##### Id #bouton-y{ display: inline-block; transition: all 0.5s; }
##### Hover .btn:hover{ background-color: #d35400 ; }
##### Before / after .btn:before{ content: ":) " ; }
##### Effet! .btn:before{ content: ""; display:block; min-height:100%; background-color:#2c3e50; }

:first-child / :last-child (ie9+)
Css
							
				.btn{
					color: white;
					background-color: orange;
				}
				.btn:first-child{
					background-color: blue;
				}

:nth-child(n) (ie9+)
Css
							
				.btn:nth-child(3){
					background-color: orange;
				}
				.btn:nth-child(2n){
					background-color: blue;
				}

Parent - enfants
Css
							
				.parent-btn .btn{
					color: white;
					background-color: orange;
				}

##Priorités
La dernière propriété est toujours prioritaire .selecteur{ color: blue; } .selecteur{ color: red; }
##Sauf si... .selecteur{ color: white !important; /* Hack, mais c'est le MAL! */} .selecteur.selecteur.selecteur.selecteur{ color: grey; /* Hack, plusieurs fois le même selecteur */} .parent .selecteur{ color: green; /* Avec parent, plus prioritaire */} #idselecteur{ color: blue; /* Id plus prioritaire que les class */} .selecteur{ color: red; /* class plus prioritaire que les tag */} button{ color: yellow; }
